## Tentang Aplikasi

Aplikasi berbasis Java memungkinkan pengguna untuk menemukan, mengunduh, dan membagikan berbagai karya visual. Pengguna juga dapat mengunggah hasil karya mereka sendiri agar dapat dilihat oleh orang lain. Dengan fitur pengelolaan sederhana, konten dapat diatur dengan rapi. Tampilan yang user-friendly membuat aplikasi ini nyaman digunakan.

---

## Fitur Utama

- Login & Register user
- Upload artwork
- Manajemen artwork
- Profile pengguna
- Role admin & user
